## üöÄ 3-STEP ACTIVATION PROCESS

### STEP 1: Update Module Cache (2 minutes)

**In Odoo:**
1. Click: **Apps** (top left menu)
2. Click: **Update Apps List** (button or option)
3. Wait for the refresh to complete
4. You should see message: "Modules updated successfully"

‚úÖ **Expected Result**: Module cache refreshed

---

### STEP 2: Upgrade the Module (5 minutes)

**In Odoo:**
1. Navigate: **Apps** (top left)
2. Search: Type "Recruitment" in search box
3. Click: "**Recruitment UAE - Retention & Follow-up**" module tile
4. Click: **"Upgrade"** button (green button on the right)
5. **WAIT** - Do not close browser or navigate away
6. Wait for: "Module successfully upgraded" message
7. Module status will show: "Activated" (green indicator)

‚úÖ **Expected Result**: Module installed and activated

---

### STEP 3: Clear Browser Cache & Verify (5 minutes)

**In Browser:**
1. Press: **Ctrl + Shift + Delete** (Windows) or **Cmd + Shift + Delete** (Mac)
2. Select: "All time" or "Since beginning of time"
3. Check: **Cookies and cached images/files**
4. Click: **Clear data**
5. Close the browser tab completely
6. Reopen Odoo in a new browser tab

**In Odoo (verify):**
1. Invoicing ‚Üí Invoices ‚Üí Any invoice
2. Scroll down to find new fields (should see: buyer_name, project_name, etc.)
3. Print ‚Üí Check for "Invoice with Deal Information" report

‚úÖ **Expected Result**: Module fully activated and visible

## üî¥ IF SOMETHING GOES WRONG

3. Refresh page (F5)
4. Fields should appear

### Report Missing

2. Refresh Odoo page
3. Navigate to Invoicing ‚Üí Invoices
