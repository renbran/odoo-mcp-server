## 🎨 Color & Icon Legend

### Status Badges
```
🟢 Green       - Active, Normal status
🟡 Yellow      - Caution, needs attention
🟠 Orange      - High priority, overdue
🔴 Red         - Critical, urgent action needed
⚫ Gray        - Inactive, archived
```

### Section Icons
```
💰  Financial/Payment information
💳  Payment tracking & receipts
📅  Dates, timeline, scheduling
👤  Candidate/Person information
⚠️  Alerts, risks, warnings
🎯  Assessment, evaluation
📞  Contact, communication
➡️  Next steps, actions
🏷️  Tags, classifications
📌  General information
📎  Attachments, links
🔔  Notifications, escalations
```

### Button States
```
[Highlighted]  - Primary action, currently available
[Normal]       - Secondary action, available
[Disabled]     - Not available in current state
[Loading]      - Processing
[Complete ✓]   - Action finished
```
