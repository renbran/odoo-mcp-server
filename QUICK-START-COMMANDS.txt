═══════════════════════════════════════════════════════════════

    python3 inspect-backend.py list uninstalled | grep -i commission

═══════════════════════════════════════════════════════════════
 STEP 5: GET MODULE DETAILS
═══════════════════════════════════════════════════════════════

On Server (Bash):

    • Name
    • State (installed/uninstalled)
    • Description
    • Dependencies
    • Version

═══════════════════════════════════════════════════════════════
 STEP 6: INSTALL MODULE (if needed)
═══════════════════════════════════════════════════════════════

⚠️  Only if dependencies are met!

═══════════════════════════════════════════════════════════════
 STEP 7: VERIFY INSTALLATION
