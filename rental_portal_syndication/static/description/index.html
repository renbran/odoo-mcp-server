What is This Module?

Like Having a Super Helper!

Imagine you have a toy that you want to sell. Instead of telling all your friends one by one, what if you could tell ONE robot, and that robot would automatically tell ALL your friends at once? That's exactly what this module does for properties (houses and buildings)!

The Magic: Add your property ONCE in Odoo, and it automatically appears on Bayut, Dubizzle, Property Finder, Houza, and more - all at the same time!

How It Works

This module extends (adds more power to) your existing Rental Management system. Think of it like adding rocket boosters to a bicycle - the bicycle still works, but now it can do amazing new things!

What It Does:
  • Connects your Odoo to property websites (portals)
  • Automatically sends property information to these websites
  • Collects interested buyers/renters (leads) from all websites
  • Keeps track of everything in one place

Getting Started

Before You Start: Make sure you have the Rental Management module already installed and working. This module builds on top of it!

Step 1: Installation

1 Go to Apps: Click on "Apps" in the main Odoo menu
2 Update Apps List: Click "Update Apps List" button
3 Search: Type "Property Portal Syndication" in the search box
4 Install: Click the "Install" button and wait for it to finish

Step 2: Get Permissions

Ask your system administrator to give you the right permissions:

Permission Level What You Can Do Who Needs It
Portal User View portals, listings, and leads Everyone who needs to see the data
Portal Manager View and edit portal listings Property managers
Portal Admin Full control including API keys IT administrators only

Step 3: Find the Menu

After installation, you'll see a new menu called:

Portal Syndication

Inside it, you'll find:

  • Connectors: Where you set up portals
  • Leads: People interested in your properties
  • Sync Logs: History of all updates
  • Feed Config: Advanced settings

Setting Up Your First Portal

Creating a Portal Connector

1 Navigate: Go to Portal Syndication â†’ Connectors
2 Create New: Click the "Create" button
3 Fill in Details:
  • Name: Give it a friendly name like "My Bayut Connection"
  • Portal Code: Choose from: Bayut, Dubizzle, Property Finder, Houza, Property Monitor
  • Company: Your company name (usually auto-filled)
  • Sync Frequency: How often to update (start with "Manual Only" while testing)
4 Security Token: The system creates a special password automatically - keep it safe!
5 Feed URL: You'll see a special web address - this is what you give to the portal
6 Save: Click "Save" button
Pro Tip: Start with just ONE portal (like Bayut) to learn how it works. Once you're comfortable, add more portals!

Sharing Your Feed URL

Each portal needs your special Feed URL to get your property listings:

1 Open your portal connector (the one you just created)
2 Copy the "XML Feed URL" field
3 Send this URL to your portal account manager
Keep It Secret! This URL contains your security token. Don't share it publicly or post it online!

Publishing Your First Property

From Property Details

1 Open Property: Go to your property in Rental Management
2 Look for New Buttons: You'll see two new smart buttons at the top:
  • Portals: Shows which portals this property is on
  • Leads: Shows interested customers
3 Portal Syndication Tab: Click on the new "Portal Syndication" tab
4 Add Portal: Click "Add a line" to add the property to a portal
5 Select Portal: Choose which portal (like Bayut)
6 Status: Set status to "Published" when ready
7 Save: Click "Save"
Auto-Magic: Once saved, the property will automatically sync to the portal based on your sync frequency!

Troubleshooting Common Issues

Issue: Can't See Portal Syndication Menu

Problem: After installing, the menu doesn't appear.

Solution:

  1. Log out and log back in
  2. Check if you have the right permissions (ask your admin)
  3. Refresh your browser (press F5)
  4. Clear browser cache (Ctrl+Shift+Delete)

Issue: Smart Buttons Not Showing on Property

Problem: "Portals" and "Leads" buttons missing from property form.

Solution:

  1. Make sure the module is fully installed
  2. Go to Settings â†’ Technical â†’ Views
  3. Search for "property.details.form.portal.buttons"
  4. If it doesn't exist, reinstall the module

Issue: Feed URL Shows "Unauthorized"

Problem: When accessing feed URL, get "Unauthorized" message.

Solution:

  1. Check that the URL includes "?token=..." at the end
  2. Verify the token hasn't been changed
  3. Make sure you're using the exact URL from the connector
  4. Try regenerating the token (in portal connector settings)

Issue: Can't Add Property to Portal

Problem: Error when trying to add property to portal list.

Solution:

  1. Check that the portal connector exists and is active
  2. Make sure the property isn't already on that portal
  3. Verify you have "Portal Manager" or "Portal Admin" permissions

Issue: Module Won't Install

Problem: Installation fails or shows errors.

Solution:

  1. Make sure "rental_management" module is installed first
  2. Check server logs for detailed error messages
  3. Verify all required modules are available: website, portal, mail
  4. Ask your system administrator for help

Frequently Asked Questions

Can I use this without rental_management?

Answer: No. This module extends rental_management and requires it to work. Think of it like needing a bicycle before you can add training wheels!

How many portals can I connect?

Answer: As many as you want! You can connect to all 5 supported portals (Bayut, Dubizzle, Property Finder, Houza, Property Monitor) plus custom ones.

Is my data safe?

Answer: Yes! Each portal connection has its own security token. Only people with the exact feed URL (including the token) can access your data.

Can I remove a property from one portal but keep it on others?

Answer: Absolutely! Just go to the property's "Portal Syndication" tab and change that portal's status to "Disabled" or delete the line.

Why does the feed URL return "not_implemented"?

Answer: This is version 0.1, and the XML feed generator is still being built. The URL works (good for testing!), but real data will come in version 0.2.

What happens if I change a property price?

Answer: In future versions, the new price will automatically update on all connected portals during the next sync. For now (v0.1), you need to update each portal manually.

Can I test this safely?

Answer: Yes! Set sync frequency to "Manual Only" and create test portal connectors. Nothing will be sent automatically. You're in complete control!

Do I need special portal accounts?

Answer: Yes, you need to register with each portal (Bayut, Dubizzle, etc.) and get their approval to list properties. This module just makes managing those listings easier!

What's the difference between Portal User and Portal Admin?

Answer: Portal User can VIEW everything. Portal Admin can also SEE and CHANGE secret things like API keys and tokens. Only trusted people should be admins!

Can I customize which fields get sent to portals?

Answer: Not yet! This feature is planned for version 0.3. Currently, all standard property fields are included.
