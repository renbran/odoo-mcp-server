RECOMMENDED WORKFLOW
====================

1. QUICK EXPLORATION
   Use Python Inspector:
   
   python3 inspect-backend.py list uninstalled
   
   This shows all modules ready to install.

2. DETAILED ANALYSIS
   Check each module:
   
   python3 inspect-backend.py info sale
   python3 inspect-backend.py info purchase
   
   This shows dependencies and details.

3. SAFE INSTALLATION
   Install in order:
   
   python3 inspect-backend.py install base_modules
   python3 inspect-backend.py install sale
   
   Each command handles dependencies automatically.

4. VERIFICATION
   Confirm installation:
   
   python3 inspect-backend.py list installed | grep sale
   
   Verify the module state.

GETTING STARTED NOW
===================

STEP 1: Test Python Inspector
  python3 inspect-backend.py list
  
  Expected: Lists all modules (installed + uninstalled)
  If error: Check server connection in .env file

STEP 2: Check Available Modules
  python3 inspect-backend.py list uninstalled

STEP 3: Get Module Details
  python3 inspect-backend.py info module_name
  
  Expected: Shows dependencies, description, version
  Check: Verify module is installable

STEP 4: Install Module
  python3 inspect-backend.py install module_name
  
  Expected: "[OK] Installation complete" message
  Verify: Run "info" command again to confirm state

TROUBLESHOOTING
===============

Problem: Connection failed
  Check: .env file has correct credentials
  Verify: Server is accessible at URL

Problem: Module not found
  Check: List modules to see actual names
  Note: Names are lowercase with underscores

Problem: Install fails
  Check: Module dependencies are installed first
  Verify: Module is installable (not broken)
  Review: Logs in /var/odoo/scholarixv2/logs

Problem: Permission denied
  Check: Running as correct user (odoo)
  Fix: Use "sudo -u odoo" prefix
