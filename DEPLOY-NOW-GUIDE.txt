================================================================================
TROUBLESHOOTING
================================================================================

If password doesn't work:
  - Verify SSH credentials in .env file
  - Try manual SSH commands above
  - Contact server admin for password reset

If "sshpass not found":
  - Install via: choco install sshpass -y
  - Or use manual SSH commands with interactive password prompt

If installation fails:
  - Check logs: tail -100 /var/log/odoo/odoo-server.log
  - Verify files copied: ls -la /var/odoo/scholarixv2/extra-addons/
  - Check manifest: grep deal_tracking __manifest__.py

If you need to rollback:
  ssh root@139.84.163.11
  systemctl stop odoo
  cp -r backups/commission_ax_YYYYMMDD_HHMMSS/* \
        extra-addons/odooapps.git-68ee71eda34bc/commission_ax/
  systemctl start odoo
